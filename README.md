# Telegram-бот "Лидогенератор"

Полностью рабочий Telegram-бот для сбора и обработки заявок на основе aiogram 3.x.

## 🚀 Быстрый старт

1. **Получите токен бота** от [@BotFather](https://t.me/BotFather)
2. **Узнайте свой ID** через [@userinfobot](https://t.me/userinfobot)
3. **Настройте окружение**:
   ```bash
   cp .env.example .env
   # Заполните BOT_TOKEN и ADMIN_ID
   pip install -r requirements.txt
   python main.py
   ```

## 📁 Структура проекта


## ✨ Основной функционал

### 📋 Пошаговая форма (FSM)
1. **Имя** - текстовая валидация
2. **Телефон** - строгая валидация (10-15 цифр, форматирование)
3. **Услуга** - выбор из 4 вариантов (inline-кнопки)
4. **Сводка** - подтверждение и отправка

### 🚀 Интеллектуальные возможности
- **FSM Context** - корректная обработка прерываний
- **Валидация полей** - регулярные выражения для телефона
- **Форматирование** - красивый вывод для админа
- **Логирование** - полная запись всех действий
- **Graceful shutdown** - корректное завершение работы

### 🛡️ Безопасность
- Проверка прав доступа для `/stats`
- Экранирование Markdown/HTML
- Защита от SQL-инъекций
- Обработка ошибок Telegram API

## 🔧 Технические детали

### Стек технологий
- **Python 3.10+**
- **aiogram 3.x** (актуальный синтаксис)
- **SQLite** (легко заменяется на Google Sheets)
- **python-dotenv**
- **PEP 8**, type hints, чистая архитектура

### FSM-состояния
```python
FormStates:
- name     # Запрос имени
- phone    # Валидация телефона  
- service  # Выбор услуги
- summary  # Финальная сводка
```

### Валидация телефона
- Регулярные выражения для проверки
- Поддержка международных форматов
- Авто-форматирование: `+79001234567` → `+7 (900) 123-45-67`
- Обработка крайних случаев

## 📊 База данных

### SQLite структура
```sql
CREATE TABLE leads (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    name TEXT NOT NULL,
    phone TEXT NOT NULL,
    service TEXT NOT NULL,
    user_id INTEGER NOT NULL
);
```

### Переход на Google Sheets
Для замены SQLite на Google Sheets:
1. Установить: `gspread google-auth`
2. Настроить service account
3. Заменить методы в `database.py`
4. Добавить в `.env`: `GOOGLE_SERVICE_ACCOUNT_KEY=...`

- `/start` - начать填写 формы
- `/stats` - статистика заявок (только админ)
- `/cancel` - отменить текущую форму

## 🔧 Особенности реализации

### Ограничения Telegram
- Бот не может писать первым
- Админ должен сначала написать боту для получения user_id
- Добавлена проверка и логирование ошибок отправки

### Обработка ошибок
- Graceful shutdown при KeyboardInterrupt
- Логирование всех ошибок
- Пользовательские сообщения об ошибках
- Автоматическое восстановление после сбоев

### Масштабируемость
- Чистая архитектура с разделением ответственности
- Легкая замена SQLite на Google Sheets
- Модульная структура для расширения функционала
